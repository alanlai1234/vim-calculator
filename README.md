# vim-calculator

a simple calculator inside vim
## Installation

use your package manager
```vimscript
Plug 'alanlai1234/vim-calculator'
```

## Usage

1. keybinding:

* default:
```vimscript
<leader>m
```
* customize:
```vimscript
nmap [key] <Plug>calc_launch
```
2. cancel input : ```<esc>```

3. equal sign not needed
